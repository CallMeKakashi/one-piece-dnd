### Skills and Tools

Skill and tool checks are weird. You can use a tool check in place of a skill check. So let's make tools more useful, and have skills and tools to interact more.
If a creature has both proficiency in a skill and a tool, it can make the tool check with advantage. This also functions with Expertise.
